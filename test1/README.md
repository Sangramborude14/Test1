# 🚀 Project Evolution: Test 1

This folder marks the beginning of the application's journey, setting up the most foundational aspects of a web server.

### ✨ What's Unique in This Version?

- **Fundamental Setup**: This is the very first step of the project, establishing the absolute basics.
- **Direct HTML Responses**: Raw HTML strings are sent directly in the route handlers using `res.send()`.
- **No External Files for Views**: Everything is handled inside the main `app.js` file, keeping it simple but monolithic.

> [!NOTE] 
> This version serves as the baseline to understand how routes operate before introducing complex routing or templating engines.
